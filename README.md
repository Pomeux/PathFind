# Pathfind

## Introduction

Pathfind est un programme pour trouver le plus court chemin de fichier .map sous différents algorithmes.

## Fonctionnalités

PathFind implémente les algorithmes suivants
- FloodFill
- Dijkstra
- A*
- Weighted A*
